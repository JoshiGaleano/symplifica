Prueba Quick - Symplifica

Usted ha sido escogido para diseñar y desarrollar un sistema que gestione los pacientes
de un hospital.

Existe un documento de requerimientos que contiene la siguiente información:
● El sistema debe asignar pacientes a una habitación
● Las habitaciones deben tener un limite de pacientes
● El sistema debe registrar pacientes si existen habitaciones con capacidad
● El sistema debe permitir crear habitaciones

Entregables:
1. Diagramas que sean necesarios para explicar su solución.
2. Patrón arquitectural, si utilizó.
3. Código que muestre su solución
Notas:

1. Un paciente tiene los siguientes campos (nombre, cédula, teléfono).
2. Una habitación tiene (número, capacidad).
3. El sistema debe estar desarrollado en PHP puro.
4. Existe una base de datos Mysql.
5. Puede crear entidades adicionales que ayuden a desarrollar el sistema.
6. Puede adicionar campos a las entidades Paciente, Habitación, Hospital.
